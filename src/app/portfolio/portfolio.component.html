Fortsaet

Fortsaet analyzes a resume and gives feedback on the different types of emotions and personality traits that the resume conveys using IBM Watson. This tool can be used to see how a resume would stack up when put through a machine learning algorithm that many companies today use. It can also be used by potential employers to help in the hiring process.
