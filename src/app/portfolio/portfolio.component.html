Upself

Upself is a web and mobile app with one goal - to make your day better. At the core of Upself is an AI powered chatbot named Upsy. Upsy can have a conversation with you, tell you a joke, or give you a compliment. Upsy is continually learning and improving to help make your day better! This project is currently in production, and is built with React for the web app, and React Native for the iOS and Android mobile app.

Quantla

Quantla is a web app that uses the Tensorflow machine learning model to make an informed decision on how and when to trade bitcoin. The app collects 142 data points every 5 minutes and uses this data to advise users whether to buy, sell, or hold. Use invite code 'matt-invite' to create an account and see how Quantla works

Fortsaet

Fortsaet analyzes a resume and gives feedback on the different types of emotions and personality traits that the resume conveys using IBM Watson. This tool can be used to see how a resume would stack up when put through a machine learning algorithm that many companies today use. It can also be used by potential employers to help in the hiring process.
